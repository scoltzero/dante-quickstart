# 确认有docker
docker --version

# 下载脚本
```
wget -qO setup_dante.sh https://raw.githubusercontent.com/scoltzero/dante-quickstart/main/setup_dante.sh
```

# 赋予脚本执行权限
```
chmod +x setup_dante.sh
```

# 执行脚本
```
./setup_dante.sh
```
# 默认账户是sockd the default username is sockd
